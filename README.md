# GRESB Reporting Skill

A Claude Code skill for preparing GRESB Real Estate Assessment submissions, calculating scores, and generating ESG reports for institutional real estate portfolios.

## What It Does

- Assess portfolio GRESB readiness and estimate scores
- Build data collection plans for energy, GHG, water, and waste
- Draft GRESB narrative sections and investor-facing ESG reports
- Identify scoring gaps and prioritize improvement actions
- Guide green star rating strategy

## Commands

| Command | Description |
|---------|-------------|
| `/assess` | Assess GRESB readiness and estimate score for a portfolio |
| `/data-collection` | Build a GRESB data collection plan |
| `/report` | Draft GRESB narrative sections and ESG report |

## Installation

```bash
npx skills add soapboxbuild/gresb-reporting-skill
```

## Coverage

- Management component: leadership, policies, reporting, risk management
- Performance component: energy, GHG, water, waste data completeness
- Like-for-Like boundary analysis
- Green Star rating guidance (1-5 stars)
- Asset types: Office, Retail, Industrial, Residential, Hotel, Diversified
