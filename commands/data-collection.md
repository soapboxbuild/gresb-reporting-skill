Create a GRESB data collection plan:

1. Map all assets by type and landlord/tenant control split
2. Identify data sources for each utility type (energy, water, waste)
3. Flag assets with data gaps
4. Recommend: green lease clauses, smart meters, utility aggregators
5. Create a data collection timeline for the GRESB submission window
6. Draft tenant data request template
